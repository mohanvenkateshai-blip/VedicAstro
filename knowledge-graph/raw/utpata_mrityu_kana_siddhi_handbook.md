 Source trail & scope
 
 
 
 This handbook follows the Utpātādi-yoga-catuṣṭaya rule: for each weekday, four consecutive nakshatras produce Utpāta, Mṛtyu, Kāṇa, and Siddhi, respectively.
 The clearest classical source trail is Muhūrta Cintāmaṇi, in the Śubhāśubha-prakaraṇa. The text gives the seed nakshatras for Sunday onward as Viśākhā, Pūrvāṣāḍhā, Dhaniṣṭhā, Revatī, Rohiṇī, Puṣya, Uttaraphālgunī; from each seed, the next four nakshatras become Utpāta, Mṛtyu, Kāṇa, and Siddhi.
 Important: This is a weekday + Moon nakshatra yoga. “Planet” here means the lord of the weekday: Sun for Sunday, Moon for Monday, Mars for Tuesday, Mercury for Wednesday, Jupiter for Thursday, Venus for Friday, Saturn for Saturday.

 How to apply this in a panchang
 
 
 1. Identify the Vāra
 Use the weekday as per the local panchang convention. In many traditional panchangs, Vāra is sunrise-to-sunrise rather than midnight-to-midnight, so local sunrise matters for edge cases.
 2. Identify the Moon’s Nakshatra
 The yoga changes when the Moon changes nakshatra. If the nakshatra changes mid-day, the Vāra–Nakṣatra yoga changes at that transition.
 3. Match the row
 Find the weekday row and compare the current nakshatra against the four cells: Utpāta, Mṛtyu, Kāṇa, or Siddhi.
 
 
 4. Interpret conservatively
 Siddhi is supportive, but it is not a blank cheque. For serious muhurta selection, still examine tithi, tara bala, chandra bala, lagna, yoga, karana, weekday defects, rāhu kāla/yamaganda, and the purpose of the activity.
 Working rule: Avoid Utpāta, Mṛtyu, and Kāṇa for major auspicious beginnings unless a qualified tradition gives a specific exception or parihāra. Prefer Siddhi only after other core muhurta factors are acceptable.

 Source notes embedded for reference
 The page is self-contained, so these source notes are included here for future recall:
 
 Muhūrta Cintāmaṇi lists the Utpātādi-yoga-catuṣṭaya seed nakshatras from Sunday onward and then explicitly expands each weekday into Utpāta, Mṛtyu, Kāṇa, and Siddhi combinations.
 The same source says the results of these four yogas are to be understood according to their names.
 For practice, use this table as a screening layer, not the sole muhurta judgment.
