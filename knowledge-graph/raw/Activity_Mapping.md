> **Source texts:** B.V. Raman's *Muhurtha (Electional Astrology)*, *Muhurta Chintamani*, *Brihat Parasara Hora Sastra* (BPHS), *Phaladeepika* (Mantreswara), *Gochar Phaladeepika* (Pulippani)

## Scoring Methodology

Every activity is scored on a 0–100 scale using these factors:

| Factor | Weight | Effect |
|--------|--------|--------|
| Prescribed Nakshatra (Moon in) | +14 | The day's Moon occupies a star classically prescribed for the activity |
| Contraindicated Nakshatra | −16, cap ≤42 | The Moon occupies a star explicitly avoided — hard contraindication |
| Favoured Weekday | +8 | The weekday's planetary lord supports the activity's nature |
| Avoid Tithi (Rikta 4,9,14; Parva 8,12,15,30) | −10, cap ≤68 | Inauspicious tithi — soft contraindication |
| Favoured Karana | +8 | The half-tithi karana aligns with the activity |
| Ravi Yoga active | +6 | Sun→Moon inclusive distance qualifies |
| Siddhi Yoga (Amrit/Sarvartha) | +8 | Vara–Nakshatra auspicious coincidence |
| Tara Balam (birth chart) | +10 / −14, cap ≤55 | Personal nakshatra compatibility with transit Moon |
| Chandra Balam (birth chart) | −10, cap ≤60 | Moon in 6/8/12 from Janma Rashi |
| Muhurta Lagna strong | +8 | Well-placed ascendant at the chosen window |
| Muhurta Lagna weak | −10, cap ≤62 | Weak ascendant at the chosen window |
| Gochar (transit) from Janma Rashi | −30 to +30 | Each planet's transit house quality (good/bad/worst) |
| Dasha context (prediction engine) | −12 to +13 | Current Mahadasha + Antardasha lord beneficence |

## General Rules (All Activities)

| Rule | Description | Source |
|------|-------------|--------|
| Rikta Tithis (4, 9, 14) | Avoid for ALL auspicious beginnings. Best for cleansing, endings, demolition. | BPHS, Raman |
| Parva Tithis (1, 15, 30) | Avoid for marriage, conception, construction. Amavasya (30) avoided for all muhurtas. | BPHS |
| Ashtami (8) | Avoid for surgery, shaving, wedding. | Raman Ch.IX |
| Dwadashi (12) | Restrict for wedding. | Raman Ch.IX |
| Janma Nakshatra | Avoid for surgery, shaving, medicine. Allowed for education, house entry, upanayanam. | Raman |
| Waning Moon (Krishna Paksha) | Preferred for surgery, debt payment, selling, resignation, harvesting. | Gochar PD |
| Waxing Moon (Shukla Paksha) | Preferred for beginnings, investments, purchases, marriage, construction. | Gochar PD |
| Abhijit Muhurta | The 8th day-muhurta (~midday). Void on Wednesday. | BPHS |
| Brahma Muhurta | 2 muhurtas before sunrise. Universal auspicious window. | BPHS |
| Disa-Shoola | Each weekday has a forbidden travel direction. Check before journeys. | Raman Ch.XIV |
| Tara Balam | Janma (1), Vipat (3), Pratyak (5), Naidhana (7) stars from birth star are unfavourable. | Gochar PD Ch.24 |
| Chandrabala | Moon should NOT be in 6th, 8th, or 12th from Janma Rashi for important acts. | Gochar PD Ch.25 |
| Muhurta Lagna | 8th house unoccupied; benefic in kendra (1,4,7,10); fixed signs preferred. | Raman |
| Sade Sati | Saturn in 12/1/2 from Janma Rasi: moderate to severe caution. | Gochar PD Ch.26 |
| Gochara Vedha | Good transit of a planet cancelled by another planet in a specific vedha house. | Gochar PD Ch.22 |
