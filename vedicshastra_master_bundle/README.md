# VedicShastra AI - Master Prompt & Asset Bundle

This bundle contains everything you need to start building **VedicShastra AI** — a state-of-the-art Vedic Astrology platform.

## What's Included

- `Master_Prompt_VedicShastra_AI.md` — The complete updated Master Prompt with all 13 chunks (including the new UI Design Module).
- `DESIGN.md` — The production-ready design system file for all UI work.

## How to Use

1. Read the `Master_Prompt_VedicShastra_AI.md` first.
2. Use the 13 chunk prompts to delegate work to agents (ruflo swarm or Claude Code).
3. Start by setting up your project using the free stack guide.
4. Copy `DESIGN.md` into your project root.
5. Use the database schema and RBAC example as foundation.
